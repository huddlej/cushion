# Cushion

A Django application for browsing and editing CouchDB documents.

 - Server
   - Databases
      - Documents
        - browse raw documents
        - create
        - edit
          - determine form from data types
          - determine form from predefined Django form
        - copy
        - delete
      - Views
        - edit original document
      - Lists
        - browse views
      - Shows
        - browse lists
        - edit original document
      - Changes
